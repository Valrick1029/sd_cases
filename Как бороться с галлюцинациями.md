Prompt engineering
1. Явно указать: "Не знаешь, скажи, не выдумывай"
2. Можно потребовать цитирование источника
3. Указать: сначала рассуждение, потом вывод
4. Стараться формулировать запрос однозначно, чтобы возможность понять неправильно стремилась к минимуму

Еще:
1. Ограничение домена, то есть предоставление базы знаний и дообучение (RAG схема)
2. Можно подобрать оптимальное значение температуры, чтобы ответы были более строгими
3. Использовать вторую модель для валидации ответов первой
4. Использовать структурированный вывод JSON
